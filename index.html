OpenFusion

An entirely free game engine, made for performance and an alternative to Clickteam Fusion.


Why OpenFusion?

OpenFusion is multi-platform website game engine that's very light-weight. The program is designed to very light-weight and efficent, so that no package manager are needed for the program; it's also blazingly fast when you're trying to compile your script with the Dwarf compiler. The entire codebase is open-source, so everyone can use/distribute it for free.

    Getting Started

    To start installing OpenFusion, you need to install these dependencies first, here's what you'll need.

  • Git - Version control
  • NodeJS - JavaScript runtime

      • Windows

      First, you need to install Git from this link, if you've not installed already.

      And then install NodeJS with this link; after you've installed these dependencies, you can now use the Git to clone the repository to your desktop folder, by using the command below inside your Windows Command Prompt (cmd.exe).

      Windows Command Prompt

      C:\Windows\System32> git clone https://github.com/openfusion-engine/openfusion.git OpenFusion

      After the you've cloned the repository, you may now see the OpenFusion in your desktop folder. Now it's the last part, moving the Dwarf compiler to the Program Files folder. After everything is done, open index.html inside the OpenFusion folder and enjoy creating your application!

      Unix-like (Linux, BSD & MacOS)

      First, you need to install Git from this link, if you've not installed already.

      And then install NodeJS with this link; after you've installed these dependencies, you can now use the Git to clone the repository to your desktop folder, by using the command below inside your terminal.

      Shell

      user@host:~$ git clone https://github.com/openfusion-engine/openfusion.git OpenFusion

      After the you've cloned the repository, you may now see the OpenFusion in your desktop folder. Now it's the last part, moving the Dwarf compiler to the binaries folder like this command below:

      Shell

      user@host:~$ sudo mv dwc /usr/bin

      It'll ask you for permission first but it's ok; and after everything is done, open index.html inside the OpenFusion folder and enjoy creating your application!

    Classic Hello, World Application

    If you've never heard about "Hello, World" program before, it's just a program displaying the word "Hello, World" on the screen. In OpenFusion, you'll need to use the Dwarf language and the OpenFusion editor to get the work done.

    First, click the new application on the menu, then click create; after that you'll go inside the editor. After you're in the editor, open click the script editor and paste the code below.

    MyApp.dw

    #!/usr/bin/dwc moduleFrom "./MyApp.json" importTo "./MyApp.js" importFrom $MyString as string when IS LOADED locate $MyString center center size $MyString auto auto setAttribute $MyString "font" "monospace" setAttribute $MyString "size" 32 setAttribute $MyString "text" "Hello, World!" show $MyString end
